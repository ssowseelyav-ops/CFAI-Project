**Smart Health prediction System Using ML**


*Abstract*

The Medical Diagnosis Reasoning Engine is an Artificial Intelligence-based project designed to assist in preliminary disease prediction using probabilistic reasoning. The system analyzes user-provided symptoms and estimates the likelihood of possible diseases by applying a simplified form of Decision Tree Algorithm . Unlike traditional rule-based systems that depend only on fixed conditions,     this project handles uncertainty by considering the probability of symptoms associated with different diseases.

The objective of this project is to demonstrate how AI techniques can support decision-making in situations where information is incomplete or uncertain. The system accepts symptoms such as fever, cough, headache, and fatigue as input and evaluates their relationship with diseases such as flu, cold, and malaria to generate probable outcomes. Additionally, the system provides explainable results by showing the reasoning behind recommendations, improving transparency and understanding.

This project serves as an educational model for understanding probabilistic inference and decision-making in Artificial Intelligence. It is intended for academic purposes and does not replace professional medical diagnosis.
